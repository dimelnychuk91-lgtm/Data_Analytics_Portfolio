# AI Era: How Tech Skills Market Changed 2022–2025

I wanted to understand what actually changed in tech after ChatGPT launched. 
So I combined Google Trends, salary data, and Stack Overflow survey to find out.

## Data
- Google Trends via pytrends — 6,300 rows
- Salary records — 71,393 rows
- Stack Overflow Developer Survey 2025 — 49,191 rows

## Stack
Python (pytrends, pandas) → Tableau Public

## What I found
- ML Engineers median salary: $189K. AI Engineers grew fastest: +59% since 2022
- 73% of AI/ML Engineers use AI tools daily — no surprise there
- Front-End devs are most worried about AI replacing them (15.8%)
- DevOps engineers couldn't care less (10%)
- Daily AI users actually earn *less* than those who don't use it — still not sure what to make of that
