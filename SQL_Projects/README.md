# sql-user-engagement-analysis

Я проаналізував, як користувачі взаємодіють із платформою та емейл-розсилками. Головна мета: виявити найактивніші ринки та оцінити якість бази користувачів для маркетингової стратегії.
Технічний стек (SQL BigQuery)Запит демонструє роботу з реальними бізнес-даними та використання просунутих інструментів:
CTE (Common Table Expressions): розбив логіку на окремі блоки для метрик акаунтів та емейлів, що робить код чистим.
Віконні функції: використав SUM() OVER для розрахунку часток по країнах та DENSE_RANK() для точного ранжування ринків.
Обробка даних: агрегація через UNION ALL, робота з типами даних (CAST) та часовими інтервалами.
Join Logic: поєднання таблиць (LEFT JOIN) для збереження повної воронки взаємодії.
Що вдалося з'ясувати. Географія: США — абсолютний лідер ринку (Rank 1), за ними стабільно йдуть Індія та Канада.
Ефективність розсилок: середній Open Rate становить 35.5%, що є дуже високим показником і свідчить про релевантність тем листів для юзерів.
Якість бази: 71.6% користувачів верифікували свої акаунти, а 82.9% аудиторії залишаються лояльними та підписаними на новини.
Динаміка: пік активності припав на грудень 2020 – січень 2021 року, після чого спостерігається стабілізація трафіку.
